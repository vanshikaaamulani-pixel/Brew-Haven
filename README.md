# Brew Haven
A modern and responsive coffee shop website built using HTML, CSS, and JavaScript.
# About the Project
Brew Haven is a web development project designed to provide an engaging online experience for coffee lovers. The website features a clean user interface, responsive design, smooth navigation, and visually appealing sections that showcase the café's offerings and brand story.
# Features
Responsive navigation bar
Interactive hero section
About Us section
Custom styling with CSS variables
Smooth and modern UI design
Floating hero image animation
Social media links
Fully responsive layout
# Technologies Used
- HTML5
- CSS3
- JavaScript
- Google Fonts
